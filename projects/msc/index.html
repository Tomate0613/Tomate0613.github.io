mc-server-control

Simple web UI for managing Minecraft servers

Features

  • Start and stop Minecraft servers
  • Access to Minecraft server console
  • Manage Minecraft server plugins/mods
  • Manage Minecraft server configuration

Software

  • Vanilla
  • Fabric
  • Paper

Installation

  1. Download latest release
  2. Install nodejs
  3. Run npm i

Usage

  • Run node server.js or npm start
